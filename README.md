# MSCS_634_Lab_6

## Purpose
The purpose of this lab is to apply frequent itemset mining techniques using Apriori and FP-Growth algorithms on a transactional dataset.

## Key Insights
- Certain items frequently occur together, indicating strong buying patterns.
- FP-Growth algorithm performed faster than Apriori.
- High lift values revealed meaningful product associations.

## Challenges
- Data cleaning and transformation into transaction format.
- Selecting appropriate support and confidence thresholds.
- Handling computational complexity in Apriori.

## Conclusion
This lab demonstrates how association rule mining can help businesses understand customer purchasing behavior and improve decision-making.
